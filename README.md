## E-Commerce Sales Analysis & Customer Insights

### Overview
Analyzed Brazilian e-commerce data to uncover sales trends, customer behavior, and payment type trends.

### Dataset
Brazilian E-Commerce Public Dataset by Olist (Kaggle)

### Tools
- Python
- Pandas, NumPy
- Matplotlib, Seaborn

### Key Insights
- Some states generate more revenue than others by a good margin
- Credit cards are the most common payment method

### Conclusion
This project demonstrates end-to-end data analysis skills including data cleaning, EDA, and business insights.
